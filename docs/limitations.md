# Limitations

- The curated dataset is an academic, source-labeled dataset built for this
  application; it is not exhaustive official source coverage.
- The sample/recovery load path reloads the canonical corpus; it is not a
  separate demo dataset.
- Text similarity is a reading aid, not a legal determination.
- The overlap signal is an exploratory similarity-based signal, not a legal
  judgment.
- Candidate application areas are deterministic rule-based suggestions and
  should be treated as candidate application context.
- The Genetic Algorithm compares grouping settings only when grouping quality
  can be evaluated; it does not optimize commercial outcomes.
- Small datasets can reduce keyword-evidence quality, grouping stability,
  related-patent density, and graph usefulness.
- Strict thresholds can produce no related-patent links or no overlap signals.
- Live EPO, USPTO, and TURKPATENT API integration is not implemented in this
  version.
